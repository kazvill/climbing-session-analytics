# Climbing Session Analytics

A web-based analytics platform for tracking and analysing climbing session performance.

## Problem
Climbers often log sessions but struggle to understand progress, weaknesses, and performance trends.

## What this app does
- Tracks climbing session data
- Analyses performance over time
- Highlights strengths, weaknesses, and fatigue patterns

## Tech stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (FastAPI)
- Database: SQL (PostgreSQL / SQLite)

## Current status
- Frontend scaffold deployed
- Backend scaffold in progress

## Planned features
- Session logging
- Performance dashboards
- Basic authentication
